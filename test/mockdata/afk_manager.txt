<a name="Information"></a><font size="4">Information</font><br />
<br />
This plugin has been designed to manage AFK players, it has a General AFK check along with a Spawn AFK check. It checks player commands for button changes and mouse movements to determine if a player is &quot;AFK&quot;. If the player is not &quot;typing/chatting&quot;, pressing any buttons or performing any movements of their mouse, they are deemed &quot;AFK&quot; and the plugin can move and/or kick them after a specified amount of time. It will warn players who are detected as &quot;AFK&quot; before being moved or kicked based on specific variables in 5 second intervals.<br />
<br />
This plugin has been designed to utilize a translation file for use of other languages.<br />
<br />
<br />

<a href="http://www.sourcemod.net/downloads.php" target="_blank" rel="nofollow noopener">SourceMod v1.7.0+</a> (Latest version 4 branch)<br />

<li>sm_afk_admins_immune - Should admins be immune to the AFK Manager? [0 = DISABLED, 1 = COMPLETE IMMUNITY, 2 = KICK IMMUNITY, 3 = MOVE IMMUNITY]</li>

<li>sm_afk_move_spec - Should the AFK Manager move AFK clients to spectator team? [0 = FALSE, 1 = TRUE, DEFAULT: 1]</li>

<li>sm_afk_kick_warn_time - Time in seconds remaining, player should be warned before being kicked for AFK. [DEFAULT: 30.0 seconds]</li>
<li>sm_afk_spawn_time - Time in seconds (total) that player should have moved from their spawn position. [0 = DISABLED, DEFAULT: 20.0 seconds]</li>
<li>sm_afk_spawn_warn_time - Time in seconds remaining, player should be warned for being AFK in spawn. [DEFAULT: 15.0 seconds]</li>

<li>sm_afk_move_warn_unassigned - Should the AFK Manager warn team 0 (Usually unassigned) players? (Disabling may not work for some games) [0 = FALSE, 1 = TRUE, DEFAULT: 1]</li>
<li>sm_afk_buttons - How many button changes should the AFK Manager track before resetting AFK status? [0 = DISABLED, DEFAULT: 5]</li>

</ul><a name="Plugin Developers"></a><font size="4">Plugin Developer Information &amp; Examples</font><br />
<br />
If you are a plugin developer who wants to add mod specific or server specific functionality to the AFK Manager or to be able to detect AFK status in your own plugin you can now use the Native and Forwards I have created to do so. If you want a generic AFK feature to be added which would be beneficial to any mod, post in the thread so I can review.<br />
<br />
Information can be found in the afk_manager.inc and the features are currently implemented in the latest release, I believe the functions are about 80% complete, so be aware they could change in the future. I am open to any feedback on whether you want to get more information from the AFK Manager or whether you think any of the current functions should change.<br />
<br />
<font size="2"><b>Example Plugins for Developers</b></font><br />
<br />
I have written a couple of small example plugins for Counter-Strike/CS:GO to show a few methods of how you can use the AFK Manager natives and forwards:<br />
<br />
<b>Example Plugin #1</b><br />
<br />
This plugin does not use any of the forwards: <a href="http://afkmanager.dawgclan.net/scripting/afk_manager_cstrike.sp" target="_blank" rel="nofollow noopener">http://afkmanager.dawgclan.net/scrip...ger_cstrike.sp</a><br />
<br />
The plugin was based off RedSwords original AFK Bomb plugin (<a href="https://forums.alliedmods.net/showthread.php?t=152329" target="_blank" rel="noopener">https://forums.alliedmods.net/showthread.php?t=152329</a>).<br />
<br />
In the example you can see how I check for &quot;Active&quot; Terrorist players (Not marked as AFK for even 1 second) in the AFK_RandomBomb() function using AFKM_IsClientAFK().<br />
<br />
You can also see how I have created a timer for any players with the Bomb and take action after a certain amount of AFK time in the Timer_CheckPlayer() function using AFKM_GetClientAFKTime().<br />
<br />
Please note while I was making this I found there was a bug in the CS_DropWeapon() function for CS:GO where the toss was not working which has been fixed in release 1.8.0.6009 and 1.9.0.6101, thanks to the #SourceMod guys for their input help and fixes.<br />
<br />
<b>Example Plugin #2</b><br />
<br />
This plugin is more specific to CS:GO around Team Swapping Issues/Modifications: <a href="http://afkmanager.dawgclan.net/scripting/afk_manager_csgo_teamchange.sp" target="_blank" rel="nofollow noopener">http://afkmanager.dawgclan.net/scrip..._teamchange.sp</a><br />
<br />
In this example you can see how I use the &quot;AFK&quot; forward events for &quot;afk_move&quot; and &quot;afk_kick&quot; to mark the player as &quot;Moved&quot; so that we can allow them to join a team again straight away by bypassing the &quot;Pending&quot; team change mode that Valve has implemented.<br />
<br />
I have also on request added a couple of additional options to:<br />
<br />
Provided a variable to override the pending team change for all players irrespective of whether they were moved by the AFK manager.<br />
<br />
Provided a variable to block players from changing directly between Terrorist and Counter-Terrorist Teams.<br />
<br />
Again these plugin are examples, there are no current translations or anything to notify players about the actions taken in these plugins.<br />
<br />
If people are interested in me enhancing these to actually be a full blown sister plugin with certain features or want to take the projects on yourself let me know.<br />
